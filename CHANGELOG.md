# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2024-04-28

### Added

- Comprehensive project documentation (README, CLAUDE.md)
- Status line component with 5-row display showing git state, context usage, session cost, and more
- Initial PyPI package structure with entry points
- Basic component discovery and installation system
- Type hints throughout codebase
- Ruff linting configuration (E, F, I, UP rules)

### Fixed

- Component installation and uninstallation logic

## [0.1.0] - 2024-04-28

### Added

- Initial project setup with basic structure
- Component manifest system (component.json)
- CLI for discovering and installing components
